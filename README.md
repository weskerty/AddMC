# Add Server Minecraft Bedrock

## Cambiar a tu Server

Clona el Respositorio, luego ve a `index.html` y edita estos:

```js
const SN1='Casita Server';      // nombre de tu server
const SU1='mc.cheagana.com';    // direccion/host ip de tu server
const SP1='19132';              // puerto
const SD1='Servidor Minecraft Bedrock'; // descripcion 
const WA1='595972184435';       // numero wpp para el boton de comunicacion en caso de que logre ver la pagina, ya que inicia MC al instante.
```

## Luego ve a ajustes de tu Fork, Pages, Elegis main y Guardar. Luego la pagina estara funcionando y puedes revisar.
Luego puedes usar cualquier acortador web para compartir
